# Git 的使用

推荐使用命令行，而不是 GUI

安装工具

- [Git](https://git-scm.com/) or [Git Bash](https://gitforwindows.org/)
- [Visual Studio Code](https://code.visualstudio.com/)
- [Git Graph](https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph)

### 克隆项目

```bash
# 克隆项目，包括子项目
git clone xxxxx --recursive
```

### 合并代码

```bash
# 切换到合并的分支
git checkout master
# 将 dev 分支合并到 master 分支
git rebase dev
```

### 提交代码

```bash
# 将要提交的文件添加到工作区
git add ./
git commit -m "feat: 新加了 xxxxx"
```

### 拉取代码

```bash
git stash -u
git pull --rebase
git stash pop
```

### 回滚操作

1. 硬回滚

```bash
git reset <hash> --hard
```

2. 软回滚

```bash
git reset <hash>
```

3. 回滚某个 commit

```bash
git revert <hash>
```
