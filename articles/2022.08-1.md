#### 封装常用库

封装是为了统一的入口和出口，方便一键更改

通常，这些都跟业务无关，尽量保持功能单一职责

- HTTP
- 拦截器（请求拦截器/路由拦截器/路由鉴权等）
- 加密/解码
- 常用函数（基本函数，使用 lodash-es）
- 公共组件（不涉及任何业务代码）
  - 弹窗
  - 页面

#### 代码

- 使用 esmodule 而不是 commonJS（注意兼容性）

  - import 和 require

- 使用可被 transpile 和 polyfill 的特性

- 根据环境，尝试使用新特性（postcss / css var / flex）
